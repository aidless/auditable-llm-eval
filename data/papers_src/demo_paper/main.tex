\section{Method}
We minimize the loss $L(\theta) = \frac{1}{n}\sum_i \ell(f_\theta(x_i), y_i)$.
The gradient is given by
\begin{equation}
\nabla_\theta L = \frac{1}{n}\sum_i \nabla_\theta \ell(f_\theta(x_i), y_i)
\end{equation}
We apply Sinkhorn-Knopp to project $B$ onto the Birkhoff polytope.
